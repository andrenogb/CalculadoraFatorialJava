Calculadora de Fatorial em Java
Este programa calcula o fatorial de um número inteiro fornecido pelo usuário, mostrando o processo da multiplicação na tela.

Funcionalidades:

Calcula o fatorial de números grandes usando o tipo long.

Exibe o passo a passo do cálculo (ex: 5x4, 20x3...).

Menu para repetir o cálculo ou encerrar o programa.

Conceitos Aplicados:

Estrutura de repetição For com decremento (n--).

Uso de variáveis acumuladoras para armazenar o resultado.

Tipo de dado long para suportar valores maiores.

Entrada de dados com Scanner e repetição com Do-While.
